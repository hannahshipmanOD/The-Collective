## What this skill does

Helps with data queries, metric pulls, and analytics for CX, TC, and sales support operations. Knows the data models and can write BigQuery SQL.

## Workflow

1. Understand the question: what metric, what time range, what granularity?
2. Load relevant data model (`DATA_MODEL_OPERATIONS.md` or `DATA_MODEL_COMMON.md`)
3. Write and explain the query
4. Interpret results if data is provided
5. Suggest follow-up analyses if patterns emerge

- `DATA_MODEL_OPERATIONS.md` — operations-specific tables
- `DATA_MODEL_COMMON.md` — shared/common tables
