# Linear Sync

## Trigger phrases
- "linear"
- "create issue"
- "log issue"
- "linear sync"
- "track this"
- "file a ticket"

## What this skill does

Helps triage, create, and update Linear issues for CX, TC, and sales support operations. Knows Opendoor's Linear team structure.

## Workflow

### Create issue
1. Understand the problem: what happened, what's the impact, what team owns it?
2. Draft issue: title, description, priority, labels
3. Create via Linear MCP tool
4. Return issue link

### Triage issues
1. List open issues for the relevant team
2. Suggest priority adjustments
3. Identify blockers

### Sync from meeting
1. Take action items from a meeting
2. Create Linear issues for each
3. Assign to correct owners

## Issue templates

### Bug / incident
```
Title: [system]: [brief description]
Description:
  ## What happened
  [description]

  ## Impact
  [who is affected, how many]

  ## Steps to reproduce
  [if applicable]

  ## Expected vs actual
  [what should happen vs what does]
```

### Process improvement
```
Title: [process]: [improvement description]
Description:
  ## Current state
  [how it works today]

  ## Desired state
  [how it should work]

  ## Why this matters
  [impact, who benefits]
```
