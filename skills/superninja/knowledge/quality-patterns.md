# Quality Patterns

Quality and review patterns for CX/TC/Sales Support.

## CX quality signals
- Recontact rate >15% → dig into root cause
- CSAT <4.5 for an XA for 3+ days → 1:1 coaching
- SLA breach rate >10% → check staffing and queue routing
- Same ticket category spiking → likely systemic issue, file Linear

## TC quality signals
- Transaction stuck in same stage >5 days → check in
- Fall-through rate spike → review deal characteristics
- Close date push >3 times → escalate to Matt
- TC with >20% at-risk rate → capacity or skill issue

## Review cadences
| Review | Frequency | Owner |
|--------|-----------|-------|
| CX metrics | Daily (standup) | Matt |
| TC pipeline | Daily (standup) | Matt |
| CSAT review | Weekly | Matt |
| Escalation log | Weekly | Matt |
| Roster health | Bi-weekly | Matt |
| Training check-in | Monthly | Matt |
